## 1. Purpose and Responsibilities

A thin styling wrapper around `@radix-ui/react-label`'s `Root` primitive. Renders a `<label>`-equivalent element with the kit's semantic token styling (uppercase, tracked, accent-colored text) and forwards every prop and behavior of the underlying Radix primitive, including its `htmlFor`/`asChild` support and its click-forwarding-to-control behavior.

**Out of scope for this component:** any field-level validation state or error message rendering (no `error`/`required`-styling props exist); associating itself with a specific input beyond the native `htmlFor` mechanism.

## 8. BDD Scenarios

```
When it mounts
```

### Associated-disabled state

```
Given a Label rendered after a sibling input with the peer class and the disabled attribute set
When it renders
Then the label shows cursor-not-allowed and opacity-70 via the peer-disabled: CSS variant
```

### Keyboard / click-to-focus behavior

```
Given a Label with htmlFor="email" wrapping no children input, and a separate <input id="email">
```
