## 1. Purpose and Responsibilities

Renders a centered empty-state placeholder (`<div>`): an icon glyph or node, an uppercase title, optional description text, and an optional action slot. Applies the TUI visual language through two `cva()`-driven variant groups â€” `size` (padding scale) and `bordered` (dashed border on/off) â€” and derives icon sizing from `size === "sm"`.

**Out of scope for this component:** deciding when to show an empty state (consumer decides based on data/query state), the action's behavior (the `action` slot renders whatever `ReactNode` the consumer passes, e.g. a `Button`), loading/error distinct visuals (this is exclusively the "no data" presentation).
